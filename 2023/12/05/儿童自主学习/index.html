培养有主动性的孩子课程学习笔记
Cemon_Liu Lv5
  2023-12-05 10:07:50   2023-12-09 11:43:22  
  • 知行合一
  

让孩子快速写作业

  1. 第一个任务20s快速启动，任务小，能量低
    1. 化大为小
    2. 从容易的开始
    3. 加入静心（写作业前的准备，小启动）
  2. 用清单法搞定顺序，释放大脑
    1. 先做简单 喜欢的
    2. 顺序基本固定
    3. 孩子自主决定顺序
    4. 孩子自己整理清单
  3. 用打勾法给大脑提供及时奖励
    1. 给于言语鼓励”哇 ，完成了，真好“
    2. 非言语：肯定，欣赏
    3. 不要建立作业成就感与 游戏，看小说之间的链接

小动作太多

  1. 大脑听不懂别，不字

  2. 说来，快点写

    1. 嗯，怎么样
    2. 顺利么
    3. 动作靠近，注视

  3. 桌面管理，移除干扰的东西

  4. 暂停法 在游戏和作业中间，插入其他事情

考试拿不到高分

  1. 大量刷题会变笨 120%刷题合适
    1. 无效重复
    2. 自动化，不思考
    3. 没法突出重点
  2. 一万小时理论，刻意练习
    1. 有目的 ：哪里不会，哪里容易出错
    2. 大量：写单词（3遍不出错），错题 （会做会讲会变形）， 卷子（划掉绝对会的）。
  3. 通过训练，终结马虎。
    1. 马虎指向结果， 无用。认真指向结果， 也无用
    2. 怎么做？ 指向过程
    3. 方法
      1. 落题 查题号
      2. 读错题干 标注 批划
      3. 算错数 口算练习，注意力练习（各种题型刷一刷， 会检查）
  4. 转换信念
    1. 错误：坏事？ 指向结果
    2. 好事：过程 提高 （做了什么样的改变 导致这个结果）
    3. 发现错误 等同于找到盲点，可以提高
    4. 为啥错，难在哪，怎么改

碰到难题就逃避

  1. 无助感导致 目标做不到导致习得性无助
  2. 儿童注意力20分钟 上限
  3. 方法
    1. 小步子原则 （10->15， 只读->思考，找笔->学习准备好）
    2. 成长性思维（有天赋，** 找办法**）
    3. 哪不会，有什么办法，从哪学

如何鼓励孩子

  1. 夸错了，指向结果，品质
    1. 真棒 好，得意洋洋无法复制
    2. 不好，沮丧退缩，不会改，装作不在乎
  2. 正确的方法
    1. 聚焦具体行为 不要抽象， 从具体转为可信再提升做的好
    2. 错题整理（有心眼） 举一反三（聪明）走神拉回来（真专心）
    3. 我看到。。。 我观察到。。。 我注意到。。。
    4. 强化 行为–>内在好处 不是为了别人 为了自己
    5. 别人家的孩子 ==>偷学** 模仿卓越**

营造好的学习氛围

氛围是一种场力，同频共振（镜像神经元，复制别人的行为）

  1. 提供榜样的作用 展示 为了一件事努力认真的样子 看书要记要看，讲解出来

  2. 静心的气场 学习的时候，家长也要同步专注

  3. 整洁便利的环境

  4. 建立情感条件反射 学习绑定打骂吼进而产生坏情绪，应该绑定好情感（爱，关心，轻松喜悦）

  5. 看到长远，放下焦虑。关于学习的重点

    1. 自主管理，强化孩子做的好负责的地方
    2. 爱学习
    3. 会学习

孩子假期任务安排

  1. 目标宏达，笼统 基本上都无法完成，能达到的目标才有效
  2. 现实性目标，计算可用于学习的时间 SMART法则
  3. 父母管 => 孩子管 使用契约法 列清单来执行，发现父母和孩子的共同需求，完成双赢

  1. 省下的时间做什么? 历练优势->点亮动力->学习的长远意义

孩子自主管理作业

  1. 推孩子就像推车，当孩子长大了，车子变大了，家长就推不动了。
  2. 管理权还给孩子，低年级父母管，高年级自己管。放手前，提前规划，
  3. 给于足够的支持
    1. 做的好 行为导向好处，及时强化
    2. 做的不好 哪不好？如何做好？如何做到？
  4. 双轨制，孩子管家长督（是否做到，如何调整），孩子应做的事情：
    1. 列清单- 执行清单
    2. 注意力
    3. 主动找方法
    4. 发现问题，主动调整
